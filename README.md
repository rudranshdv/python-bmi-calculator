# Python BMI Calculator

A beginner friendly python project that calculates Body Mass Index (BMI) based on a user's weight and height.

## Features
- Takes weight in kilograms
- Takes height in meters
- Calculates BMI using the BMI formula
- Displays the result rounded to 2 decimal places

## Concepts Used
- Variables
- User input
- Float data type
- Arithmetic operations
- Mathematical functions
- Formatted strings (f-strings)

## Formula Used

BMI = Weight / (Height²)

## Example Output

```text
.....BMI Calculator.....

Enter your weight(kg): 70
Enter your height(meters): 1.75

BMI is 22.86
```
